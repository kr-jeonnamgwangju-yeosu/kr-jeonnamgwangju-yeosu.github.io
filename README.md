# 여수시 · 전남광주통합특별시

> 여수시 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-jeonnamgwangju-yeosu.github.io/**

여수시는 한반도 남해안 중앙부에 돌출한 여수반도와 365개의 크고 작은 섬으로 이루어진 다도해의 항구도시로, 옛 전라남도 동부권을 대표하는 해양·관광·산업도시다. 동서남북 사방이 바다에 열린 리아스식 해안을 따라 879km에 이르는 긴 해안선이 펼쳐지고, 온난한 해양성 기후 덕에 사시사철 여행객이 끊이지 않는다. 대중가요 '여수 밤바다'로 전국적 명성을 얻은 낭만의 도시이자, 임진왜란 당시 이순신 장군이 지휘한 전라좌수영의 본영이 자리했던 호국의 고장이기도 하다. 2012 여수세계박람회를 성공적으로 치른 국제 해양관광 거점이며, 여수국가산업단지를 품은 대한민국 석유화학산업의 심장부이기도 하다. 오동도의 동백숲, 돌산대교의 야경, 향일암의 일출, 그리고 밤바다를 수놓는 낭만포차가 어우러져 '한 번 오면 반드시 다시 찾는 도시'라는 별명을 얻었다.
